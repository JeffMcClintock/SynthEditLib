High Pass (TPT)

A first-order (6 dB/octave) high-pass filter built with the Topology-Preserving Transform (zero-delay feedback). The cutoff is prewarped (g = tan), so it stays accurate right up to Nyquist.

Pins

  • Signal - the audio input to be filtered.
  • Pitch - the cutoff frequency (1 Volt/Octave).
  • Output - the high-pass output (6 dB/octave).
